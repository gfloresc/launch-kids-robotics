# 🚀 LAUNCH Robotics Course for Homeschoolers 🤖

Welcome to the official repository for the LAUNCH Robotics Course for Homeschoolers at First Baptist Church Laredo.

This course introduces students to the exciting world of robotics, programming, electronics, sensors, and autonomous systems through hands-on activities and beginner-friendly coding projects.

---

# 📚 Course Information

- Course: Robotics for Homeschoolers
- Program: LAUNCH Homeschool Program
- Institution: First Baptist Church Laredo

---

# 🎯 Course Objectives

Students will:

- Learn the basics of robotics and automation
- Understand how sensors and motors work
- Program robots using beginner-friendly code
- Develop problem-solving and logical thinking skills
- Explore autonomous navigation and line-following robots
- Work collaboratively in robotics challenges

---

# 🧰 Technologies and Platforms

This course may include activities using:

- BBC micro:bit
- Cutebot robots
- Block programming
- Python programming
- Sensors and actuators
- Robotics simulations
- Introductory AI concepts

# 🛣️ Line Following Track Templates

Students and parents can use the following printable templates to create custom tracks for line-following robot activities.

These patterns can be printed and taped together to build different autonomous navigation circuits and challenges for the robots.

# 📸 Classroom Philosophy

This course emphasizes:
- Learning by doing
- Creativity and exploration
- Teamwork
- Engineering thinking
- Fun and curiosity

Students are encouraged to experiment, make mistakes, and improve their designs through iteration and testing.
